# idbat-v7-mobile

Application mobile Compose Multiplatform (Android & iOS).

## Structure

- `composeApp`: Module partagé contenant le code UI Compose Multiplatform et l'application Android.
- `iosApp`: (À créer) Projet Xcode pour l'application iOS.

## Prérequis

- JDK 17
- Android Studio
- Xcode (pour le développement iOS)

## Lancer l'application Android

Ouvrez le projet dans Android Studio et lancez la configuration `composeApp`.

## Lancer l'application iOS

Ouvrez le dossier `iosApp` dans Xcode et lancez l'application.
